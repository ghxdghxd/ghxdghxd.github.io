# tips

## 多层的图例

```R
scale_colour_manual(values = c("CHB" = '#E41A1C',  "MXL" = "#377EB8", "CEU" = "#377EB8",
                    "ASW" = "#984EA3","YRI"="#FF7F00", "TCGA" = "gray"),
                    breaks = c("TCGA", "CEU", "CHB", "ASW", "MXL", "YRI"))
# values 增加多图例
# breaks 图例排序
```
