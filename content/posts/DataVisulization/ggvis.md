# ggvis

```R
mtcars %>% ggvis(~wt, ~mpg) %>% layer_points(size := 25, shape := "diamond", stroke := "red", fill := NA)

mtcars %>% ggvis(~wt, ~mpg, fill = ~factor(cyl)) %>% layer_points() %>% group_by(cyl) %>% layer_model_predictions(model = "lm") %>%
```
